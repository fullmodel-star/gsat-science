# 🧬 學測生物題庫（biogsat-qbank）

大學學測 自然考科「生物」部分的官方歷屆題庫 PWA，單檔離線可用。沿用學測英語題庫（gsat-qbank）引擎，改為生物版並擴充多選作答、圖表題、題組脈絡。

## 內容
- **題源**：大考中心 學測自然考科 官方公開試題 **111–115**（從自然合卷中抽出生物題）＋ **翰林 學測精彩解析**（免費公開）逐題詳解。
- **題數**：練習題 71（單選 30＋多選 41）＋ 非選／混合題 7 ＝ **78 題**。
- **每題**：題幹＋選項＋答案＋章節＋主題＋難易度＋逐題詳解；圖表題內嵌官方原圖；題組題附共用引文與共用圖。
- **答案品質**：已對大考中心官方答案交叉核對（0 誤）。

## 功能
- 整卷模擬（依年度・計時）／章節練習（細胞・遺傳・演化・延伸）／多選題特訓／每日一練／錯題本／收藏／弱點診斷／混合題非選瀏覽。
- 深色模式、考試倒數、PWA 可安裝、離線可用、進度存本機。

## 技術
- 單檔 `index.html`（題庫內嵌於 `const QB`）。
- localStorage 主 key：`biogsat_qb_v1`；SW 快取：`biogsatqb-v1`；引導命名空間：`wc.biogsat-qbank`。（皆與英語家族分開，不互相污染）
- 主題色 `#059669`（生物綠）。

## 資料重建
題庫由 PDF 自動抽取建置：官方試卷解析(骨架)＋翰林解析(答案/章節/詳解)以題號合併，111 舊版以內容相似度對接；圖表題與題組圖以 PyMuPDF 裁切官方 PDF。
