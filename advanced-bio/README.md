# 🧪 分科生物題庫（fenke-qbank）

大學分科測驗「生物考科」官方歷屆題庫 PWA，單檔離線可用。與學測生物／地科同一套引擎、相互獨立。屬「加深練習」——分科考選修生物Ⅰ-Ⅳ，範圍比學測廣。

## 內容
- **題源**：大考中心 分科測驗 生物考科 官方公開試題 **112–114**（3 年）＋ **翰林 分科測驗精彩解析**（免費公開）逐題詳解。
- **題數**：練習題 116（單選 77＋多選 39）＋ 混合題/非選 30 ＝ **146 題**。
- **章節**：選修生物Ⅰ 細胞與遺傳・Ⅱ 植物・Ⅲ 動物構造與生理・Ⅳ 生態與演化・生物(全)基礎。
- 圖表題內嵌官方原圖（裁切至答案行前、不洩答）；答案為官方公布。
